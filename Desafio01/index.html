O que é HTML?

HTML (HyperText Markup Language) é a linguagem base para páginas da web, estruturando texto, imagens e links.

Tags Mais Usadas
  • <html>, <head>, <body> — Estrutura básica
  • <h1> a <h6> — Títulos e subtítulos
  • <p> — Parágrafos
  • <strong> — Texto em negrito
  • <abbr> — Abreviações com título
  • <blockquote> — Citações
  • <hr> — Linha horizontal
  • <ul> e <li> — Listas
  • <a href=""> — Links
  • <img src="" width=""> — Imagens
  • <input type="text" /> — Campo de texto
  • <input type="number" /> — Campo numérico
